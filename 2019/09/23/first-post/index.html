建站札记

发表于 2019-09-23 更新于 2019-10-29 分类于 日常

“1. 世界是所有实际情况。”

——维特根斯坦《逻辑哲学论》

我其实挺不擅长表达的。开一个博客，其实也是试试而已，随心情而建。连这第一篇博文都是建站好久以后才写的。

所以，也没想过有什么独立域名，也没有独立的服务器。用的是 GitHub Pages 的服务。

这个网站名字，最后是以 io 结尾，看起来也挺古怪的，这是因为以 io 结尾的域名是不要钱的。

也许有些同学好奇，建站用的是什么技术呢？其实是 Hexo 这个框架 + Markdown。

建一个像这种博客网站非常水，没啥技术难度。你需要做的，就是把文档的入门篇看完，然后自己折腾。过程中，不懂的地方看文档，然后试验几下，就成了。

对于网页前端，我也并未有什么深刻认识或者研究，所以，现在就是这样，一个轻量级的框架，加上一堆杂乱的〈thoughts(碎碎念)〉1。

不过，我也有一些要求，这个博客要能排版数学公式，毕竟如果我哪天有什么想法，想写篇关于理学的文章出来，公式怎么办？

MathJax 完美解决了这个问题2，Markdown 也有支持，所以，这个问题，也没花多久就解决了。

熟识命令行的同学，这其实只有几句话而已，也就不多废话了。

最后，依照传统，

\[ \text{Hello \LaTeX} \]

于是，就可以用 LaTeX 来打数学了。

神說、要有光、就有了光。

——《创世纪》1:3

  1. 这里尝试了一下 ruby 标记。这样用好像有些不对……↩︎

  2. 数学公式用 KaTeX 渲染其实也可以，不过，MathJax 功能更多一些，不过 MathJax 有一点实在有点坑，就是网页加载完，它才开始渲染公式。↩︎
